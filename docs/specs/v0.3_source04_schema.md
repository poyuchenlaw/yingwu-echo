## §一 Schema 表單

| 表名 | 說明 |
|------|------|

| scenes | 場景記憶錨點 |

| player_ratings | 五維評分 |

## §二 煉化機率系統

L1 普通: writing_chars >= 50 -> 100%
L2 精怪: 3 same-species + writing_chars >= 200 + emotion_tags >= 3 -> 30%
L3 神獸: 3 same-species rare + writing_chars >= 500 + active_days >= 5 -> 10%

## §三/§四 戰鬥 State Machine

IDLE -> SUMMONED -> MIRROR_WINDOW_OPEN -> CAPTURED
                                       -> RETURNED_TO_OWNER

五行剋制: x1.35 攻 / x0.85 守
永久映刻: 不歸還，imprinted_from_player_id 欄位記錄來源玩家
落後反轉: defender HP < 30% -> 觸發第二次 MIRROR_WINDOW_OPEN
玩家技能: 不直接傷害怪獸（TODO: 技能表 v0.4）

## §五 Enum 清單

- wuxing: metal / wood / water / fire / earth
- three_phase: biomorphic / tectonic / phantasmal
- rarity_tier: common / rare / legendary
- triangle_pos: attacker / defender / support
- battle_state: idle / summoned / mirror_window_open / captured / returned_to_owner
